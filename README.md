# 📝 Simple Todo App (Spring Boot + React + ShadCN + Tailwind CSS)

This is a simple full-stack **Todo application** built using:

- ⚙️ Spring Boot (Java)
- ⚛️ React.js
- 💅 ShadCN UI + Tailwind CSS
- 🔗 Axios for frontend-backend communication

---

## 📌 Features

- ✅ Create a new todo
- 📋 View all todos
- ❌ Delete a todo by ID

---

## 🛠 Tech Stack

### Backend:
- Spring Boot (REST API)
- Java 17+
- Maven
- MySQL 

### Frontend:
- React.js
- ShadCN UI components
- Tailwind CSS
- Axios

---

## 🚀 Running the Project

### Backend (Spring Boot)

1. Clone the repo and navigate to the backend directory.
2. Configure your `application.properties` with DB credentials.
3. Run the project via your IDE or:

```bash
mvn spring-boot:run
